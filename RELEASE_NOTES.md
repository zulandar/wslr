## What's New

- Initial release of WSLR - WSL Instance Manager
- System tray integration with ~20MB memory footprint
- Distribution management (start, stop, restart, terminate)
- Real-time resource monitoring (CPU, memory, disk per distribution)
- Install distributions from Microsoft Store catalog
- Clone existing distributions
- Export/import distributions
- Pin favorite distributions

## Requirements

- Windows 10 version 2004+ or Windows 11
- WSL 2 enabled
- .NET 8.0 Runtime
